# LanKontroler
## Opis
Aplikacja napisana na potrzeby obsługi sterownika Lan Kontroler 3.5 sterującego grzałką CWU.
## Technologie
> Android Studio  
> TinyControl Lan Kontroler 3.5  
> HTTP GET/POST  

## Funkcjonalność
- włączanie/wyłączanie grzania
- tworzenie harmonogramów
- archiwizacja danych
- zadawanie temperatury
